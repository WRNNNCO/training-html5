Bref historique

  • 1991 HTML
  • 1994 HTML 2
  • 1996 CSS 1 + JavaScript
  • 1997 HTML 4
  • 1998 CSS 2
  • 2000 XHTML 1
  • 2002 Mise en forme sans tableau
  • 2004 WHATWG
  • 2005 AJAX
  • 2007 Le W3C récupère les travaux du WHATWG
  • 2009 HTML 5 + CSS 3

Contexte

  • Nouveaux usages : blog, ecommerce, vidéos, ...
  • Plus d'interactivité native (sans plugin)
  • La réponse : HTML5 + API JavaScript + CSS 3

HTML 5

  • <!DOCTYPE html>
  • Nouvelles balises : section, article, aside, nav, footer, header, hgroup, figure, time, ... (exemple)
  • Balises vidéos et audio (exemples audio et video)
  • Nouveaux champs de formulaire : date, email, URL, search, number, regex, ... (exemple)
  • Nouveaux attributs notamment autofocus, placeholder
  • Certaines balises et attributs dépréciés : frame, frameset, big, font, center, acronym, ...
  • Intégration de SVG (exemple, autre exemple)
